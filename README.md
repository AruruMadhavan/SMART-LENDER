## How to run (step by step)

1. Install dependencies:
   ```
   pip install -r requirements.txt
   ```

2. (Optional вЂ” a dataset is already included) Regenerate the dataset:
   ```
   cd data
   python3 generate_dataset.py
   ```
   > **Important:** this sandbox has no internet access, so a *synthetic*
   > dataset (same columns as the real Kaggle "Loan Prediction" dataset) was
   > generated instead. For your final submission, download the real dataset
   > from Kaggle ("Loan Prediction Problem Dataset") and replace
   > `data/loan_data.csv` with it вЂ” no other code needs to change.

3. Run EDA + preprocessing + train all 4 models + save the best one:
   ```
   cd ../notebook
   python3 smart_lender_pipeline.py
   ```
   This prints accuracy for Decision Tree, Random Forest, KNN, and XGBoost,
   saves 6 charts to `notebook/charts/`, and saves the best model to `models/`.

   > This sandbox couldn't install `xgboost` (no internet), so the script
   > automatically falls back to `GradientBoostingClassifier` (same
   > boosted-trees idea). It auto-detects xgboost вЂ” once you `pip install
   > xgboost` on your own machine and rerun, it'll use the real XGBoost.

4. Run the web app:
   ```
   cd ../app
   python3 app.py
   ```
   Open **http://127.0.0.1:5000** вЂ” fill the form в†’ get
   "Loan Approved" / "Loan Rejected" with a confidence %.
